# LearnUp - Platform Pembelajaran Modern 🎓

Platform interaktif untuk meningkatkan _skill_ teknologi yang dilengkapi dengan Landing Page responsif dan Admin Dashboard fungsional.

## Fitur Utama

- **Desain Modern & Responsif:** Mendukung tampilan mobile dan desktop secara sempurna dengan sentuhan estetika premium (Glassmorphism & Animasi).
- **Dark/Light Mode:** Terintegrasi di Landing Page dan Dashboard Admin dengan memori sesi lokal (`localStorage`).
- **Registrasi Instan:** Formulir _waitlist_ yang otomatis masuk ke Dashboard Admin tanpa _backend_ eksternal.
- **Admin Dashboard Terproteksi:** Memerlukan login password untuk mengakses data pengunjung. Mendukung Export ke CSV/JSON dan manajemen tabel penuh.

## Struktur Direktori

- `index.html` : Halaman Utama (Landing Page)
- `admin.html` : Panel Dashboard Admin
- `css/` : Menyimpan skema gaya (_styling_) aplikasi
- `js/` : Menyimpan logika interaksi (_scripting_)

## Teknologi yang Digunakan

- HTML5, CSS3 murni, dan Vanilla Javascript.

## Deployment

Situs ini dioptimalkan untuk berjalan di platform _hosting_ statis seperti [Netlify](https://www.netlify.com/) atau [GitHub Pages](https://pages.github.com/). Pastikan Anda menargetkan file `index.html` sebagai _entry point_.
